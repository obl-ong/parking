# `parking.obl.ong`

This repo contains the source code for https://parking.obl.ong, the free domain parking service from Obl.ong.

## How to use

Point your domain's DNS at this:

| Type | Name | Value | TTL |

| --------------- | --------------- | --------------- | --------------- |

| CNAME | @ | parking.obl.ong | 300 |
